# Program

This folder is the top-level home of MacroIR / MacroDR program identity.

It is distinct from:

- `projects/`, which holds bounded efforts
- `papers/`, which holds publication-facing outputs
- the code baseline under `include/`, `src/`, `tests/`, `tools/`, and `legacy/`
- `theory/`, which will hold baseline and exploratory theory

## Current contents

- `PROGRAM.md`
  - human-readable program definition
- `PROGRAM.yaml`
  - machine-readable program manifest
- `notes/`
  - preliminary program vision and strategic framing documents
- `source-notes/`
  - raw transcripts and other source material pending analysis
- `archive/`
  - cross-cutting historical program material

## Governance note

The canonical machine-readable program manifest is:

- `program/PROGRAM.yaml`

The copy under `projects/program-project-framework/PROGRAM.yaml` should be
treated as transitional project-local governance material rather than the
canonical program manifest.

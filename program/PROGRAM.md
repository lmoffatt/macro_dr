# MacroIR / MacroDR Program

## Purpose

MacroIR / MacroDR is a long-lived research and engineering program for the
development, validation, application, and communication of recursive Bayesian
inference methods for kinetic and related models.

## What belongs to the program baseline

The program baseline is the current accepted state of:

- code
- theory
- canonical documentation

In this repository, that baseline is currently distributed across:

- `include/`, `src/`, `tests/`, `tools/`, `legacy/`
- `docs/`
- selected theory documents that will be migrated into `theory/`

## Main workstreams

- theory

## Current structural status

The repository is transitioning from a mixed historical layout toward a
program-oriented structure with explicit homes for:

- program identity

# Program/Project Framework

This project defines the vocabulary, manifests, and organization rules used to
manage the MacroIR / MacroDR program and its subprojects.

Primary document:

- `PROJECT.md` - project definition, scope, deliverables, validation, and exit
  criteria
- `VOCABULARY.md` - canonical definitions of program/framework terms
- `LIFECYCLE.md` - stages, checks, readiness, and validation model
- `ORGANIZATION.md` - proposed repository structure and document splitting rule
- `PROJECT.yaml` - draft machine-readable manifest for this project
- `PROGRAM.yaml` - draft machine-readable manifest for the MacroIR program

Intended outputs:

- canonical vocabulary for program/project/baseline/paper/run/artifact/archive
- initial `PROGRAM.yaml` and `PROJECT.yaml` templates
- lifecycle and validation model
- repository organization proposal for `macro_dr`

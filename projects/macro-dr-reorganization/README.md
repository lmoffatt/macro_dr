# MacroDR Reorganization

This project applies the Program/Project Framework to the `macro_dr`
repository.

Primary documents:

- `PROJECT.md` - purpose, scope, deliverables, checks, and exit criteria
- `PROJECT.yaml` - machine-readable project status and checks
- `DECISIONS.md` - accepted reorganization decisions and pending curation choices
- `CLASSIFICATION.md` - current repository materials classified by framework role
- `TARGET.md` - repo-specific target structure for `macro_dr`
- `MIGRATION.md` - phased migration plan from current layout to target layout
- `REMAINING.md` - closeout items still pending after the main migration work

Intended outputs:

- classification of current repository materials by category
- target structure for `macro_dr` that respects the current code tree
- migration plan from current layout to the target layout
- identification of active, canonical, and archival materials
- initial placement rules for program, theory, code, projects, and papers

# eLife 2025 Execution Project

This project is the execution and reproducibility workspace for the current
MacroIR validation-paper effort.

Primary materials:

- `PROJECT.md` - purpose, scope, deliverables, checks, and exit criteria
- `PROJECT.yaml` - machine-readable project status and checks
- `ops/` - local execution scripts
- `figures/` - analysis and figure-generation notebooks/scripts
- `runs/` - recorded runs and provenance
- `data/` - models and experiment inputs
- `archive/debug/` - retained debug and scratch artifacts moved out of the root

This project is distinct from:

- `papers/macroir-elife-2025/`, which is the publication-facing paper pack
- `projects/macrodr-program-roadmap/`, which owns the program-level roadmap

# MacroDR DSL

This project develops the MacroDR DSL as a baseline component of the program.

Primary documents:

- `PROJECT.md` - purpose, scope, deliverables, checks, and exit criteria
- `PROJECT.yaml` - machine-readable project status and checks
- `INVENTORY.md` - current DSL pain points and acceptance cases
- `DSL_RECONSTRUCTION.md` - current reconstructed DSL model
- `DSL_CHANGES.md` - delta from older DSL docs to the current model

Expected outputs:

- clearer and more usable DSL behavior for active workflows
- DSL changes that support better diagnostics, validation, and scripting
- a project home for bounded DSL work rather than scattering it across ad hoc fixes

# MacroDR Diagnostics

This project improves diagnostics and error reporting across MacroDR.

Primary documents:

- `PROJECT.md` - purpose, scope, deliverables, checks, and exit criteria
- `PROJECT.yaml` - machine-readable project status and checks

Expected outputs:

- better runtime and parser-facing diagnostics
- clearer error context for scripts such as `projects/eLife_2025/ops/local/figure_1.macroir`
- reusable error-reporting improvements for the baseline, not just one paper workflow

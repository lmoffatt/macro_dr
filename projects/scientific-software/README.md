# Scientific Software

This project tracks the scientific-software line of work inside the MacroIR /
MacroDR program.

It covers the conceptual and methodological ideas that are not specific to one
MacroIR algorithm, but that shape how the program is designed, organized,
validated, and communicated.

Primary documents:

- `PROJECT.md` - purpose, scope, deliverables, checks, and exit criteria
- `PROJECT.yaml` - machine-readable project status and checks

Expected future related material may include:

- theory promoted into `theory/scientific-software/`
- publication-facing material promoted into `papers/`
- internal subprojects or workstreams if this area later splits into several
  independent lines

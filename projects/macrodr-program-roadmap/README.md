This project defines the next program-level roadmap for MacroIR / MacroDR.

Primary documents:

- `PROJECT.md` - purpose, scope, deliverables, checks, and exit criteria
- `PROJECT.yaml` - machine-readable project status and checks

Expected outputs:

- a canonical program roadmap document
- milestone and dependency structure for the next phase of the program
- alignment between theory, code, validation, execution projects, and papers
